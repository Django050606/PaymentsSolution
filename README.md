# Payment Service Test Task

A full-stack .NET 8 application for receiving and tracking payment data.

## Features
- **Frontend**: Razor Pages with Bootstrap & jQuery Validation.
- **Backend**: ASP.NET Core Web API with a Service-based architecture.
- **Database**: MS SQL Server (LocalDB) via EF Core.
- **Security**: Simple API Key validation for payment legitimacy.

## How to Run
1. Clone the repository.
2. Ensure you have **LocalDB** installed (standard with Visual Studio).
3. Press **F5** in Visual Studio. 
4. The application will automatically create the database and launch the UI.

## API Documentation
- Swagger UI is available at `/swagger` for testing the endpoints manually.
